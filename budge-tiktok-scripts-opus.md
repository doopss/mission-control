# Part IV: Production Quick Reference

## Pre-Production Checklist
- [ ] Script finalized with timing marks
- [ ] Text overlays written out
- [ ] Music/sound selected
- [ ] Filming location prepped (authentic mess is fine)
- [ ] Ring light or natural lighting ready
- [ ] Phone charged + storage cleared
- [ ] Budge app ready for demo

## Filming Best Practices
1. **Film vertical** (9:16 aspect ratio)
2. **Start recording BEFORE the hook** (clean edit start)
3. **Record multiple takes** of hook (this is crucial)
4. **Film app demos separately** for clean screen recording
5. **Keep natural imperfections** (relatable > polished)
6. **Maintain eye contact with lens** (connection)

## Editing Checklist
- [ ] Hook is IMMEDIATELY engaging (no slow start)
- [ ] Text overlays readable (large, contrasting)
- [ ] Music levels don't overpower voice
- [ ] Captions/subtitles added (accessibility + sound-off viewers)
- [ ] Transitions are clean (no jarring cuts)
- [ ] CTA is clear and visible
- [ ] Total runtime is appropriate (30-60s sweet spot)

## Posting Checklist
- [ ] Compelling caption written
- [ ] Hashtags added (6-8)
- [ ] Cover image selected (engaging frame)
- [ ] Link in bio is correct and working
- [ ] Posted at optimal time
- [ ] First 30 min: Active in comments
